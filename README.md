🏧 ATM Simulation Project (Python)

A simple ATM (Automated Teller Machine) simulation built using Python.
This project demonstrates basic banking operations like deposit, withdrawal, balance checking, and account management using modular programming.

📂 Project Structure
ATM_Project/
│── main.py          # Entry point of the program
│── account.py       # Handles account creation and user data
│── balance.py       # Displays account balance
│── deposit.py       # Handles deposit operations
│── withdraw.py      # Handles withdrawal operations
│── statement.py     # Shows transaction history

🚀 Features
🧾 Create and manage bank accounts
💰 Deposit money
💸 Withdraw money with validation
📊 Check account balance
📜 View transaction history (statement)
🔐 Basic input validation for safer transactions

🛠️ Technologies Used
Python 3.x
File handling / basic data structures

▶️ How to Run

1-Clone the repository:

https://github.com/aditya208146-lang/ATM

2-Navigate to the project folder:

cd atm-project

3-Run the program:

python main.py

🧠 How It Works
main.py acts as the controller and menu interface
Each operation (deposit, withdraw, etc.) is handled in separate modules
Functions are imported and executed based on user choice
Data is stored and updated during runtime (can be extended to database/file storage)
